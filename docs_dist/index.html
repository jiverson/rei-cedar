The Cedar 1 pattern library has been deprecated. This documentation is provided for legacy projects. New projects should use the docs located at rei.com/cedar.

Whats New?

These updates are provided for teams who have not yet been able to migrate to our updated version of the design system.

Cedar2 alignment

Type, buttons, and inputs have been updated to be more inline with cedar 2 design guidence.
